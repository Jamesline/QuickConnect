Overview

What is QuickConnect?

QuickConnect is an Integration framework that combines a number of pre-packaged Azure components with Cluster Reply’s best practice Integration principles and patterns. The idea behind QuickConnect is to be able to offer a fully-fledged iPaaS solution on the Azure stack addressing an important gap in the Azure Integration Services area.

Enterprise Integration is typically a long-term and strategic investment enabling organisations to evolve and grow with minimal impact to business processes. QuickConnect helps organisations achieve this by adopting best practice Integration patterns with a first Integration up and running within 2-3 weeks. ​

Technical

The Architecture

The 20 years experience and the power of Azure shines through in our architecture. Building on the excellent Azure Integration Services and adding Cluster Reply know how, we have been able to produce a highly versitile and adaptable solution for delivering cloud integration.

Our architecture delivers a number of options from classic ESB, Event driven messaging to microservice delivery. Each easily configurable before deployment. The result is a stable base for the delivery of customer solutions.
